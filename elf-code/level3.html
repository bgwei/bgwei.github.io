LEVEL 3

Constraints

Program the elf to the end goal in no more than 4 lines of code and no more than 4 elf command/function execution statements in your code.

Solution

for (var i = 0; i < 3; i++) {
  elf.moveTo(lollipop[i]);
}
elf.moveUp(1);

A for loop is used to move to each lollipop - incrementing the lollipop index every time you reach the previous lollipop works perfectly here. After reaching the last one, you need only move up 1.
