LEVEL 5

Constraints

Program the elf to the end goal in no more than 10 lines of code and no more than 5 elf command/function execution statements in your code.

Challenges

Use elf.ask_munch(0) and I will send you an array of numbers and strings similar to:

[1, 3, "a", "b", 4]

Return an array that contains only numbers from the array that I give you. Send your answer using elf.tell_munch(answer).

Solution

var munchkinQuestion = elf.ask_munch(0);
var munchkinAnswer = munchkinQuestion.filter(elem => typeof elem === 'number');
elf.moveTo(munchkin[0]);
elf.tell_munch(munchkinAnswer);
elf.moveUp(2);

Here I simply retrieved the munchkin’s input with munchkinQuestion, computed the answer by filtering the input array for elements of type “number,” and stored that answer in munchkinAnswer. Some simple movement commands and a call to give the munchkin the answer complete the level.
