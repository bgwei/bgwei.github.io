LEVEL 2

Constraints

Program the elf to the end goal in no more than 5 lines of code and no more than 5 elf command/function execution statements in your code.

Challenges

Add 2 to the returned numeric value of running the function elf.get_lever(0) .

For example, if you wanted to multiply the value by 3 and store to a variable, you could do:
var sum = elf.get_lever(0) * 3

Then submit the sum using:
elf.pull_lever(sum)

Solution

elf.moveLeft(6);
elf.pull_lever(elf.get_lever(0)+2);
elf.moveLeft(4);
elf.moveUp(10);

Again, relatively straightforward, this was an introduction to interacting with the levers. A few move commands and (elf.get_lever(0)+2) to add 2 to whatever arbitrary value you are given.
